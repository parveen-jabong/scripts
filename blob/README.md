## Blob
Blob Storage Script to compress and push the compressed files with content encoding gzip

node blobStorage ./assets/<folder> key container

```js
    Example:
    node blobStorage.js ./assets/js Moc5FT31RKywxLUhtZq2es4HkYgRIuNuOLwkpA047qr73ivO4uKbDZV1CV7uQI4P003IJhPvhMcrN+PZkCZDRQ== web-assets
```
